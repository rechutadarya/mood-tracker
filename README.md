# Daily Mood Tracker
Речута Дарья, миб-301

## Описание
Веб-приложение для записи настроения. Данные сохраняются в LocalStorage, статистика отображается через Chart.js.

## Стек
- HTML, CSS, JavaScript
- Chart.js (CDN)
Добавлена поддержка записи настроения в localStorage
 • Реализован график изменений настроения
 • Добавлены функции обновления данных и отрисовки графика
 • Улучшена структура кода

## Как запустить
1. Открыть index.html в браузере.
